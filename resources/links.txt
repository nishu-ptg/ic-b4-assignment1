Assignment -01 [ Student Management Project]
Task: 100 marks (Pass Marks: 40)


1. The Data Structure-

Each student object in your students.json file should follow this structure:

id: Unique Identifier (Integer or String)
name: Full Name
email: Valid Email Address
phone: Contact Number
status: Enrollment Status (e.g., "Active", “On Leave”, "Graduated", "Inactive")


2. Required Class Structure

Create a class named StudentManager. It should encapsulate all the logic for interacting with the JSON file.

Methods to Implement:

public function getAllStudents(): Returns an array of all student objects.
public function getStudentById($id): Finds and returns a single student.
public function create($data): Appends a new student to the file.
public function update($id, $data): Finds the student by ID and updates their details.
public function delete($id): Removes the student from the array and saves the file


3. Implementation Requirements
The Interface (UI)

List View (index.php): A clean HTML table showing all students. Add "Edit" and "Delete" buttons in an "Actions" column.
Create Form: A simple form with inputs for Name, Email, Phone, and a <select> dropdown for Enrollment Status.
Edit Page: A pre-populated form that loads the existing student data based on the id passed in the URL (e.g., edit.php?id=123).


The Backend Logic

JSON Handling: You must handle the "Update" and "Delete" by reading the entire JSON file into a PHP array, modifying the array in memory, and then overwriting the .json file with the updated array.

Data Validation: Ensure the email is in a valid format using filter_var(). Prevent duplicate IDs.
